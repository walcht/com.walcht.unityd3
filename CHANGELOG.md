### Added

  scale.
- Shape generators including: 2D line, 3D line, 2D primitive, and 3D primitive.
- Samples including: 2D barchart, 3D barchart, 2D linechart, 3d linechart,
  and 2D scatterplot.
- Basic CSV importer.
- Basic utils suck as extent calculation
